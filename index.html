Analyzing Authorship of Disputed Federalist Papers Using Unsupervised Machine Learning

Alice Liu, Ishita Abrol, Saathvik Selvan, Sushant Kannan

Abstract

This paper aims to solve the controversy of authorship for The Federalists Papers by utilizing an unsupervised machine learning algorithm for authorship attribution. To solve this classification problem, we trained a K Means model on a set of articles with known authorship. Then we compared the predicted label of the received unknown to the labels of the known documents through multiple features including lexical and syntactic features. The author can then be identified for the unknown document if the similarity level is close to each feature, which is seen in the labels.
