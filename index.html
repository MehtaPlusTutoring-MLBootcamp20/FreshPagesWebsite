Analyzing Authorship of Disputed Federalist Papers Using Unsupervised Machine Learning

Alice Liu, Ishita Abrol, Saathvik Selvan, Sushant Kannan

Abstract

This paper presents the controversy of authorship for The Federalists Papers by practicing anunsupervised machine learning algorithm for authorship attribution. To solve the classificationproblem, a method is utilized, which is to divide a plane to identify the training set of knownauthorship. Then compare the received unknown to the known documents through multiple features,specifically lexical-syntactic features. Therefore, the same author can be identified for the unknowndocument if the similarity level is close to each feature.
